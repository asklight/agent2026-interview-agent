# 题库来源与生成说明

## 数据范围

`tools/build_xiaolin_question_bank.py` 从小林 Coding 公开 sitemap 中发现 MySQL、Redis、操作系统、计算机网络和后端面试页面，并仅提取页面 URL 与标题层级作为选题索引。

生成结果：

- `docs/question-bank-sources.json`：可审计的 URL、主题与项目模块映射。
- `apps/server/src/main/resources/data.generated.sql`：1,980 张可直接初始化的结构化题卡。

## 版权与处理原则

生成器不保存或导入文章段落、图片、代码样例。每张题卡的提问、评分点、常见误区、追问和场景追问都按照本项目的面试训练格式重新组织。

## 更新题库

联网抓取并重建：

```powershell
python tools\build_xiaolin_question_bank.py
```

如果站点临时限流或网络不可用，基于已抓取的来源清单重新生成：

```powershell
python tools\build_xiaolin_question_bank.py --from-cache
```

后端在 `dev` 与 `prod` 配置中同时加载手工题卡 `data.sql` 和生成题卡 `data.generated.sql`。
